AusStage documents the performing arts in Australia. AusStage now provides audiences with the chance to contribute to this important cultural record by letting companies know what you thought of the productions.

This project is being led by Flinders University alongside the companies and artists. It has been funded by the National e-Research Architecture Taskforce (NeAT). For more information about the service see the About Page
