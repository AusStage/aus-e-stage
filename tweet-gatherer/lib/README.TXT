Ensure that the following libraries are available in this directory:

- tweetStream4J (including supporting libraries)
  - instructions for building this library are available here: http://techxplorer.com/2010/10/13/exploring-tweetstream4j-2/
- Oracle Database 11g Release JDBC drivers (11.2.0.1.0 or above) http://www.oracle.com/technology/software/tech/java/sqlj_jdbc/index.html
- Apache Commons - Codec (1.4 or above) http://commons.apache.org/codec/
- Joda-Time (1.6.1 or above) http://joda-time.sourceforge.net/
- twitter-text-java
  - instructions for building this library are available here: http://techxplorer.com/2010/08/16/exploring-twitter-text-java/
- Apache Commons - Email (1.2 or above) http://commons.apache.org/email/
- Java Mail Library (1.4.3 or above) http://www.oracle.com/technetwork/java/index-jsp-139225.html
